O **Engaja Rap Club** é um ecossistema Web3 revolucionário que transforma a interação entre artistas de rap e seus fãs através de uma economia tokenizada. Nossa plataforma combina:

* 🎵 **Rádio 24h Adaptativa** com playlist inteligente
* 💰 **Sistema de Recompensas** em tokens $BSTR
* 🎯 **Campanhas Gamificadas** para artistas independentes
* 📱 **Telegram Mini App** nativo e intuitivo
* 🔗 **Integrações Multiplataforma** (Spotify, YouTube, Instagram)

## 🚀 **INÍCIO RÁPIDO**

### **Pré-requisitos**

* Node.js 18+
* npm ou yarn
* Conta no Telegram
* Carteira TON (opcional)

### **Instalação**

```bash
# Clone o repositório
git clone https://github.com/DGuedz/ENGAJA-RAP-CLUB-TELEGRAM-MINE-APP.git
cd ENGAJA-RAP-CLUB-TELEGRAM-MINE-APP

# Instale as dependências
npm install

# Configure as variáveis de ambiente
cp .env.example .env
# Edite .env com suas chaves

# Execute em desenvolvimento
npm run dev

# Build para produção
npm run build
```

### **Variáveis de Ambiente**

```env
# Telegram
TELEGRAM_BOT_TOKEN=seu_token_bot
TELEGRAM_WEBAPP_URL=sua_url_webapp

# TON Network
TON_NETWORK=testnet
TON_WALLET_SEED=sua_seed_carteira

# Database
DATABASE_URL=postgresql://...

# APIs
SPOTIFY_CLIENT_ID=seu_id_spotify
YOUTUBE_API_KEY=sua_chave_youtube
BAASIC_API_KEY=sua_chave_baasic
```
